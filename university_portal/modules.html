CS301 • 4 Credits • Spring 2026

Module Description

This module provides a comprehensive introduction to the fundamental concepts of data structures and algorithms. Students will learn how to design, analyze, and implement efficient algorithms and data structures to solve complex computational problems. Topics include arrays, linked lists, stacks, queues, trees, graphs, sorting, and searching algorithms.
