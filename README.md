# Smart University System (SUS)

<br>

## Project Overview
Smart University System (SUS) is a web application designed to manage academic processes in a university environment. The system provides tools for students, teachers, and administrators to manage courses, grades, attendance, and academic records efficiently.

<br>

## Features
- User authentication and authorization
- Student dashboard
- Teacher dashboard
- Course management
- Grade management
- Attendance tracking
- Data visualization of student performance

<br>
